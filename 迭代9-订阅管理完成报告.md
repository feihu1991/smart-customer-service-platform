## 实现内容

### 1. User 模型扩展
在 `prisma/schema.prisma` 中为 User 模型添加了以下字段：

```prisma
planId             String?        // 当前订阅的套餐ID（关联SubscriptionPlan）
subscriptionStatus SubscriptionStatus @default(active) // 订阅状态
subscriptionExpireAt DateTime?    // 订阅到期时间
```

新增了 `SubscriptionStatus` 枚举：
```prisma
enum SubscriptionStatus {
  active      // 有效
  cancelled   // 已取消
  expired     // 已过期
  pending     // 待生效
}
```

### 2. 订阅管理 API

#### GET /api/subscription
获取当前用户订阅信息，包括：
- 用户基本信息
- 当前套餐详情
- 订阅历史（最近10条）
- 今日剩余次数

#### POST /api/subscription/upgrade
升级用户套餐，参数：
- `planId`: 目标套餐ID
- `billingCycle`: 计费周期 (monthly/yearly)

功能：
- 自动计算订阅时长
- 如果当前有有效订阅，累加时长
- 创建新的订阅记录
- 更新用户套餐信息

#### POST /api/subscription/cancel
取消用户订阅，参数：
- `cancelImmediately`: 是否立即取消

功能：
- 支持取消自动续费（保留当前套餐至到期）
- 支持立即降级为免费版

### 3. 前端订阅管理页面

新建 `/subscription/manage` 页面，包含：

#### 当前订阅状态展示
- 套餐名称和图标
- 订阅状态标签（有效/已取消/已过期）
- 到期时间显示
- 今日剩余次数
- 每日限额
- 套餐价格
- 套餐包含功能列表

#### 套餐升级入口
- 跳转至套餐选择页面

#### 订阅历史
- 显示最近10条订阅记录
- 每条记录包含：套餐、订阅时间段、价格、状态

#### 取消订阅功能
- 弹窗确认
- 支持两种模式：取消续费（到期降级）/ 立即取消
